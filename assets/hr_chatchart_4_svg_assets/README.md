# HR ChatChart / HR 話圖 SVG 素材包

本素材包包含最後一次生成方向的四款可編輯 SVG 重建版：

1. `01_round_speech_arrow.svg`：圓形對話框＋上升箭頭圖表
2. `02_soft_app_icon.svg`：柔和 App Icon 風格＋折線圖表
3. `03_waveform_to_chart.svg`：語音波形轉圖表概念
4. `04_geometric_box_chart.svg`：幾何對話框＋長條圖表

## 重要說明
- 這些檔案是重新繪製的真正 SVG 向量素材，不是將 PNG 改名成 SVG。
- 文字維持可編輯 `<text>` 物件，若開啟環境沒有中文字型，外觀可能依系統字型略有差異。
- 配色依照台積配色方向：白、黑/深灰、灰、紅。
- 文字規則：中文在英文上方；中文行 `HR` 為紅色、`話圖` 為黑色；英文行 `HR` 為黑色、`ChatChart` 為紅色。
